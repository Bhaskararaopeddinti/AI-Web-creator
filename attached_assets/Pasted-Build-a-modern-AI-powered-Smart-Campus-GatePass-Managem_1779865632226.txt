Build a modern AI-powered Smart Campus GatePass Management System for colleges and hostels with a professional dark-themed admin dashboard.

The application should manage:
- Student outgoing/incoming records
- Hostel management
- Department management
- QR-based gate pass verification
- Role-based authentication
- Real-time student tracking

Main Roles:
1. Admin
2. Student
3. Warden
4. Security Guard

-----------------------------------
ADMIN FEATURES
-----------------------------------

Create a complete Admin Dashboard with:

- Total students count
- Active outings
- Pending requests
- Late return alerts
- Live student tracking
- Analytics charts
- Notifications system
- Reports export (PDF/Excel)

Admin can:
- Add/Edit/Delete students
- Manage wardens
- Manage security guards
- Manage hostels
- Manage departments
- View all outing requests
- Reset passwords
- Control user roles
- View system logs

-----------------------------------
STUDENT PROFILE FIELDS
-----------------------------------

Each student profile must include:

- Full Name
- Email
- Registration Number
- College ID
- Roll Number
- Parent Phone Number
- Hostel Name
- Department Name
- Room Number
- Profile Photo
- User Role
- Account Status
